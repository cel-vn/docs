## ğŸ”§ Platform Features

### Documentation Management
- **Live Preview** - Real-time MDX rendering with syntax highlighting
- **AI-Enhanced Content** - Automated documentation generation from code
- **Search Integration** - Full-text search across all documentation
- **Version Control** - Git-based content management and history

### User Management
- **Role-Based Access** - Admin, member, and customer permission levels
- **Secure Authentication** - JWT tokens with OTP email verification
- **Admin Dashboard** - Complete user CRUD operations and analytics
- **Profile Management** - User preferences and activity tracking

### Technical Stack
- **Next.js 15.1.2** - React framework with App Router
- **Fumadocs** - Documentation framework with MDX support
- **TypeScript** - Type-safe development environment
- **Tailwind CSS** - Utility-first styling framework

## ğŸ¯ Getting Started

### For Developers

```bash
# Clone the repository
git clone https://github.com/cel-vn/docs.git

# Install dependencies
npm install

# Start development server
npm run dev

# Open in browser
open http://localhost:3000
```

### For Content Contributors

1. **Login** - Access the platform with your credentials
2. **Navigate** - Use the sidebar to explore existing documentation
3. **Contribute** - Create or edit MDX files in the `/content` directory
4. **Preview** - See changes in real-time with hot reloading

### For Administrators

1. **Admin Dashboard** - Access user management at `/admin/dashboard`
2. **User Creation** - Add new users with role assignments
3. **Database Viewer** - Monitor system data at `/admin/database`
4. **System Management** - Configure permissions and settings
